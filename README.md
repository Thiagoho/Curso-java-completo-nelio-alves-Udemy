## Sobre o curso

Este repositório reúne todos os exercícios, desafios e projetos desenvolvidos ao longo do curso **Java COMPLETO - Programação Orientada a Objetos + Projetos**.

Durante o curso são abordados desde os fundamentos da programação até o desenvolvimento de aplicações completas utilizando Java, Spring Boot, bancos de dados relacionais e NoSQL.

> **Curso desenvolvido por:**  
> **Nelio Alves**

---

# Principais Tecnologias

| Tecnologia | Descrição |
|------------|-----------|
|  Java 21 | Linguagem principal utilizada durante todo o curso. |
|  Eclipse IDE | IDE utilizada para desenvolver os exercícios e projetos. |
|  Spring Boot | Desenvolvimento de APIs REST e aplicações Web. |
|  MySQL | Banco de dados relacional utilizado com JDBC. |
|  MongoDB | Banco NoSQL utilizado nos projetos Spring Boot. |
|  Postman | Testes das APIs REST. |
|  Spring Data JPA | Persistência de dados utilizando ORM. |
|  Git & GitHub | Versionamento de código e portfólio. |
|  Maven | Gerenciamento de dependências dos projetos Java. |

# Objetivos de Aprendizagem

Ao longo deste curso estou desenvolvendo conhecimentos para:

-  Dominar Programação Orientada a Objetos com Java.
-  Escrever código limpo e organizado.
-  Desenvolver aplicações orientadas a objetos.
-  Trabalhar com UML.
-  Criar aplicações Desktop.
-  Desenvolver APIs REST.
-  Utilizar Spring Boot.
-  Trabalhar com JDBC.
-  Utilizar JPA/Hibernate.
-  Trabalhar com MySQL.
-  Trabalhar com MongoDB.
-  Utilizar Git e GitHub.
-  Aplicar boas práticas de desenvolvimento.
